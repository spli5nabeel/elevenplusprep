📊 CSV Import

CSV columns (in order): subject, topic, question, option_a, option_b, option_c, option_d, answer (0-3), explanation, hint, difficulty (1-3)

📖 JSON Schema Reference
{
  "subject":     "english | maths | verbal | nonverbal",   // required
  "topic":       "e.g. spelling, arithmetic, analogies",   // required
  "question":    "The question text",                       // required
  "options":     ["A", "B", "C", "D"],                     // required, 2–4 items
  "answer":      0,                                         // required, 0-indexed
  "explanation": "Why the answer is correct",              // required
  "passage":     "Reading passage text",                   // optional (comprehension)
  "hint":        "A short hint",                           // optional
  "difficulty":  2,                                        // optional, 1=easy 2=medium 3=hard
  "active":      true                                      // optional, default true
}
